GitHub repositories

Code for working with ISO/TEI-Spoken is hosted in different public GitHub repositories:

  • spotei: This is the main repository for code from the Transcription+ project. It systematically collects, catalogues, and documents tools (e.g. XSLT stylesheets) developed within various processing pipelines for large-scale corpus projects, and makes them available as a unified software library.
  • teispeechtools: An older tool library for the standard, developed by Bernhard Fisseni at IDS.
  • teilicht: Code for the TEI-Licht web services, based on teispeechtools.
  • zumultapi: The ZuMult API includes Java code for a Java object representing an ISO/TEI transcript and for indexing ISO/TEI transcripts for use with the MTAS system (CQP queries over Lucene indices).
