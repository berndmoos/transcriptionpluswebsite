TEILicht webservices

A landing page for the current version of the TEILicht web services, hosted at IDS Mannheim, is at https://clarin.ids-mannheim.de/teilicht/. You can find the corresponding sources in this GitHub repository.

The TEILicht webservices are integrated in the WebLicht tool chainer.

The Transcription+ project will revise and extend TEILicht and make the new version available via a server at AdWHH.
